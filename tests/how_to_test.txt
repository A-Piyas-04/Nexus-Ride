How to Run Tests

1. Activate the Virtual Environment
   Open a terminal in the project root (e:\Projects\NexusRide) and run:
   .\venv\Scripts\activate

2. Start the Application Server
   The tests require the server to be running on port 8001.
   Run this command in a separate terminal window:
   uvicorn app.main:app --reload --port 8000

3. Run the Test Scripts
   Open a new terminal (or use the original one), ensure the virtual environment is activated, and run the tests sequentially:

   # Test Signup
   python tests/test_signup.py

   # Test Login
   python tests/test_login.py

   # Test Subscription Creation
   python tests/test_subscription_post.py

   # Test Subscription Retrieval
   python tests/test_subscription_get.py
